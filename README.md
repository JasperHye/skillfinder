# SafeFinder: Automation-First Orchestrator

A safe, standards-compliant orchestrator skill that ensures the "Automation-First" principle. It automatically intercepts complex operational requests and recommends automation tools or scripts from the ecosystem.

## 🌟 Main Features
1. **Smart Task Orchestration**: Dynamically breaks down your intent into execution pipelines.
2. **Proactive Diagnostics**: Scans your local environment and recommends missing essential skills.
3. **Global Sourcing**: Automatically searches primary platforms (ClawHub, MCP Market, etc.) for missing capabilities.
4. **Built-in Security Audits**: Pre-evaluates target scripts for vulnerabilities and suspicious calls before prompting installation.

## 🚀 Installation
Use Clawhub to install directly:
```bash
npx --yes clawhub install skill-safefinder
```
