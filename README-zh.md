# 智能探路者 (Skill-Safefinder) 合规版

安全合规的本地自动寻源总控节点。本技能接管复杂长链路需求，贯彻“自动化优先”理念，自动帮您寻找和推荐生态脚本与扩展包。

## 🌟 主要功能
1. **智能任务编排**：自动拆解复杂动作意图，转化为严密的执行管线。
2. **主动缺失诊断**：主动扫描本地挂载环境，缺乏“必备工具”时主动推荐补齐方案。
3. **全网生态寻源**：自动爬取 ClawHub、MCP Market 等世界主流生态目录寻找极客防线。
4. **内置安全扫描**：发现外部工具后，在后台自动检索可疑特征并出具合规报告。

## 🚀 安装方式
推荐通过 Clawhub 一键安装：
```bash
npx --yes clawhub install skill-safefinder
```
